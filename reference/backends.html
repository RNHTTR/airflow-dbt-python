airflow_dbt_python.hooks.backends

Backends for storing dbt projects and profiles.

The backend interface includes methods for pulling and pushing one or many files. Internally, backends use Airflow hooks to execute the actual pushing and pulling.

Currently, only AWS S3 and the local filesystem are supported as backends.

airflow_dbt_python.hooks.backends.build_backend(scheme: str, conn_id: Optional[str] = None) → airflow_dbt_python.hooks.backends.base.DbtBackend

Build a DbtBackend as long as the scheme is supported.

A local filesystem backend.

Intended to be used only when running Airflow with a LocalExceutor.

class airflow_dbt_python.hooks.backends.localfs.DbtLocalFsBackend(*args, **kwargs)

A concrete dbt backend for a local filesystem.

This backend is intended to be used when running Airflow with a LocalExecutor, and it relies on shutil from the standard library to do all the file manipulation. For these reasons, running multiple concurrent tasks with this backend may lead to race conditions if attempting to push files to the backend.

airflow_dbt_python.hooks.backends.localfs.py37_copytree(source: StrPath, destination: StrPath, replace: bool = True)

A (probably) poor attempt at replicating shutil.copytree for Python 3.7.

shutil.copytree is available in Python 3.7, however it doesn’t have the dirs_exist_ok parameter, and we really need that. If the destination path doesn’t exist, we can use shutil.copytree, however if it does then we need to copy files one by one and make any subdirectories ourselves.

An implementation for an S3 backend for dbt.

class airflow_dbt_python.hooks.backends.s3.DbtS3Backend(connection_id: Optional[str] = None, *args, **kwargs)

A dbt backend implementation for S3.

This concrete backend class implements the DbtBackend interface by using S3 as a storage for pushing and pulling dbt files to and from. The backend relies on Airflow’s S3Hook to interact with S3. A connection id may be passed to instantiate the S3Hook.

connection_id

An optional string of an Airflow connection id to use for the S3Hook.
